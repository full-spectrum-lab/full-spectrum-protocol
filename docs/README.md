## Boundary

This directory contains public-facing protocol documents, not internal total archives.

That means:

- not every historical draft belongs here,
- not every internal engineering note should be mirrored here,
- not every philosophical material should be elevated to protocol status.

The goal is clarity, not total inclusion.
