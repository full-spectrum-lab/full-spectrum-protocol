# Protocol Public Preview v0.1.0

> Repository: `full-spectrum-protocol`  
> Date: 2026-07-09  
> Status: public protocol preview

## What this milestone means

This milestone means the repository is now structured as a public-facing protocol repository rather than a mixed archive.

It is intended to support:

- protocol reading,
- RFC review,
- schema inspection,
- example-based critique,
- future compatibility discussion.

It is **not** a claim that the protocol is complete or finalized.

## Included in this milestone

- public README entry cleanup
- bilingual entry path
- `START_HERE.md`
- documentation index
- RFC set `0001` to `0006`
- schema set for identity, capability, governance event, risk, audit, FSHI request/response, and cross-enterprise audit envelope
- mapping center and standards mapping
- examples and validation scripts

## What is still intentionally unfinished

- full permission / consent object chain
- ESS request / result schema pair
- guardian review schema
- complete Cell Protocol publicization
- protocol-network implementation
- formal conformance policy

## Why this milestone matters

Without a stable public protocol entry, external readers cannot distinguish:

- protocol core,
- local engine,
- enterprise application packaging,
- internal archives.

This milestone creates that separation.
